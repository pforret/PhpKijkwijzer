## Installation

You can install the package via composer:

```bash
composer require pforret/PhpKijkwijzer
```

## Usage

``` php
$obj = new Pforret\PhpKijkwijzer();
echo $obj->echoPhrase('Hello, pforret!');
```

## Testing

